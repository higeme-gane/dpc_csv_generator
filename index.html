📋 使い方（30秒で完了）

  1. 1 このフォームで DPC6桁コード と 集計区分 を入力
  2. 2 「GitHub Actions で CSV を生成する」をクリック
  3. 3 表示されたリンクをクリック → Actions タブ で同じ値を入力して「Run workflow」を押す
  4. 4 完了後、右下の artifact から CSV をダウンロード

※ データはリポジトリに格納済み（parquetファイルは Git LFS 使用推奨）
